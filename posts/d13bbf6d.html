NEO4J安装

发表于2023-11-27|更新于2024-03-28
|字数总计:482|阅读时长:1分钟|阅读量:|评论数:

下载

下载地址：https://neo4j.com/deployment-center/

选择版本号及你的操作系统 下载

需要注意neo4j 5对应jdk17，neo4j 4对应jdk11。对应jdk版本下，也可以重装jdk。

win R- cmd- java -version查看java版本

下载后在某路径下解压即可

配置环境变量

系统环境变量中

Path中新增一条

确定-确定-确定

启动neo4j

在bin目录下，文件路径栏输入cmd回车。（即：在该目录下，进入管理员命令模式。）

输入

neo4j.bat console

如果报错

A JNI error has occurred, please check your installation and try again.

就是neo4j和jdk的版本不对应。

显示如下则启动成功

可以在浏览器中输入localhost:7474打开。默认账号和密码均为：neo4j。首次登录会提醒修改密码。打开就能看见自己的图数据库了。

如果你跟我一样jdk用的8，需要卸载重装。

jdk卸载：

首先环境变量中删除JAVA_HOME以及Path当中的java bin及 java jre等的设置。

然后通过控制面板卸载程序删除，应该是有两个东西，JAVA SE开头和JAVA update 右键卸载。

jdk17安装：

下载地址：https://www.oracle.com/java/technologies/downloads/#jdk17-windows

下载installer一路下一步即可。

配置环境变量

配置JAVA_HOME

配置Path

如果你也需要jre

刚安装完的目录下是没有jre的，需要在路径下右键打开powershell窗口执行命令：

bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre

不会返回什么东西，然后文件里就会有jre了

如上就重装jdk完成。

新建图

在最上栏输入代码即可

如：

#增加一个节点
create (n:Person {name:'我',age:33})  #有name，age两个属性
#带有关系属性
create (p:Person{name:"我",age:"33"})-[:包工程{金额:10000}]->(n:Person{name:"包工头",age:"35"})
